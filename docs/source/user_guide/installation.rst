Installation
============

Using pip
---------

|name| is available on the |pypi_long| (PyPI), and it can be installed using the Python package installer pip_. If your have pip_ installed, you can open a terminal and run:

.. code::

    pip install aeroframe

.. seealso::

    How to get started with pip_:

    * https://pip.pypa.io/en/stable/quickstart/

From Github
-----------

Alternatively, you can clone the most recent version of |name| from Github_, navigate to the repository root folder and in a terminal run:

.. code::

    python setup.py install

Check the installation
----------------------

If |name| has been installed correctly, you will be able to call |name|'s command-line tool in a terminal. If you run...

.. parsed-literal::

     |name_cli| -h

... a short help page will show up.
